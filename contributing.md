# Contributing to This Project

Here’s how we work on this project together.

## How to Start

1. **Clone the repository** (if you haven't already):  

  ```bash
  git clone https://github.com/your-username/your-repo.git
  cd your-repo
  ```

2. Create a branch for your work
Always make a new branch for any feature or fix:

  ```bash
  git checkout -b my-feature
  ```

3. Work on your changes

    Add features, fix bugs, or update files.

    Keep your code clean and readable.

4. Commit your changes

  ``` bash
  git add .
  git commit -m "Describe what you did"
  ``` 


5. Push your branch

  ```bash
    git push origin my-feature
  ```

* Open a Pull Request (PR)

* Open a PR from your branch to main.

* Add a clear description of what you changed.

* Code Review

* We Review each other’s PRs before merging.

* Suggest improvements, discuss changes, and approve when ready.

* Merge

* Once approved, merge the PR into main.


* Pull the latest changes before starting new work:
```bash
git checkout main
git pull origin main
```
